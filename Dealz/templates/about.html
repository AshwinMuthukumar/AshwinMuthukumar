Dealz



This project is intended for making use of today’s popular technology Web Scraping.

Currently there are few websites that could provide the same functionality as our project but RTSA stands out due to it’s less time consumption and average accuracy.

By combining scraping with sentimental analysis, this project would remove the requirement to write a bulk of codes in order to get a pretty well analysis on any trend.

Dealz could be used for any analysis with 60% accuracy.

Users could analysis data for movies, healthcare, global news and other trends.This website is intended for all researchers,students and for the public for casual analysis.
